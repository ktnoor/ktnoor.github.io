Adding New Files

To add new email signature assets:

  1. Place your image or document files in the email-signature folder
  2. Access them via: {% raw %}{{ base_path }}{% endraw %}/email-signature/your-filename
  3. Supported formats: JPG, PNG, GIF, SVG, PDF

Example: If you add deakin.jpg to the folder, it will be accessible at:
{{ base_path }}/email-signature/deakin.jpg
